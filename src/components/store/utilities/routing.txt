*api/tasks/*
This part should get all the tasks from everyone.

*api/tasks/:taskId*
Gets Specific Task

*api/users*
This gives you all the users

*api/users/:userid*
This gives the specific user information

*api/users/:userId/tasks*
This gives you all the tasks from user

*api/users/:userId/boards*
Gives all Boards from User

*api/boards*
Gives you all boards

*api/boards/:boardId*
Give you the specific board

*api/boards/:boardId/users*
Gives all users in board

*api/boards/:boardId/tasks*
Gives all tasks in board (edited)
